# PLAYWRITELEARNINGS

Repository for Playwright and JavaScript learning exercises.

## Contents
- `01_basics.js` — simple console example
- `chapter_01_basics/` — chapter 1 example files

## Prerequisites
- Node.js installed (https://nodejs.org/)

## Run examples
From the project root run:

```bash
node 01_basics.js
# or run files in chapters, e.g.:
node chapter_01_basics/03_verifysetup.js
```

## Notes
- A `.gitignore` is included to ignore `node_modules`, editor settings, and Playwright artifacts.
- Repo was initialized and pushed to `origin/main`.
